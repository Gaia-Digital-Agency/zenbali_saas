Homepage: Published events are displayed. All past latest 100 events are shown.
Creator Page: By default, it shows upcoming events and events from the last 90 days. There is an option to view all events, including those older than 90 days.
Admin Page: Shows all events, including unpublished ones, and includes events that are up to 120 days in the past.
Database: There is no automatic data retention policy that deletes events after a certain period. Events are only deleted when a user (creator or admin) manually deletes them.
